# Multi-Class Emotion Classification

A simple emotion classification model using RoBERTa that can detect different emotions from text.

## How to Run

1. **Clone the repo**
   ```bash
   git clone <your-repo-url>
   cd Multi-Class-Emotion-Classification
   ```

2. **Generate the model**  
   Run the Jupyter notebook to train and save the model:
   ```bash
   jupyter notebook Multi_Class_Emotion_Prediction.ipynb
   ```

3. **Install uv**  
   Follow the instructions at [https://docs.astral.sh/uv/](https://docs.astral.sh/uv/)

4. **Run the application**
   ```bash
   uv run main.py
   ```

That's it!
